## 🤖 AI Prompt Station

### Proposal Generator
```
I need to write a freelance proposal for:
- Client: [name/company]
- Project: [describe the project]
- My skills: [relevant skills]
- Timeline: [estimated duration]
- Budget range: [if known]

Please create a professional proposal that includes:
1. Understanding of their needs (show I listened)
2. My approach and methodology
3. Deliverables with timeline
4. 3 pricing tiers (Basic/Standard/Premium)
5. Why I'm the right fit
6. Social proof section (where to add testimonials)
7. Next steps / CTA

Tone: Professional but personable
Length: 1-2 pages
```

### Pricing Calculator Prompt
```
Help me price this freelance project:
- Service type: [design/development/writing/etc.]
- Scope: [describe deliverables]
- Estimated hours: [your estimate]
- My experience level: [junior/mid/senior/expert]
- Client type: [startup/small biz/enterprise/agency]
- Market: [US/EU/global]

Please provide:
1. Hourly rate recommendation (with reasoning)
2. Project-based price (3 tiers)
3. Value-based pricing option
4. Comparison to market rates
5. Upsell opportunities to increase project value
```

### Client Communication Templates
```
Generate professional email templates for:
1. Initial inquiry response (within 2 hours)
2. Follow-up after discovery call
3. Proposal send + gentle follow-up sequence (day 1, 3, 7)
4. Project kickoff email
5. Progress update (weekly)
6. Revision request handling (scope creep prevention)
7. Invoice send with payment reminder sequence
8. Project completion + testimonial request
9. Re-engagement email (past client, 3 months later)

My name: [name]
My service: [service]
Tone: [professional/friendly/casual]
```

### Scope Creep Prevention Prompt
```
A client is asking for additional work beyond the original scope:
- Original scope: [what was agreed]
- New request: [what they're asking for]
- Relationship: [new client/long-term/VIP]

Help me craft a response that:
1. Acknowledges their request positively
2. Clearly explains it's outside scope
3. Offers it as an add-on with pricing
4. Maintains the relationship
5. Sets a precedent for future boundaries
```
