## 🤖 AI Prompt Station

### Weekly Planning Prompt
> Copy this into ChatGPT/Claude at the start of each week:

```
I need to plan my week. Here are my current priorities:
- [List your top 3 priorities]

My energy patterns:
- Morning: [High/Medium/Low energy]
- Afternoon: [High/Medium/Low energy]
- Evening: [High/Medium/Low energy]

Please help me:
1. Break each priority into daily actionable tasks
2. Schedule tasks based on my energy patterns
3. Identify potential blockers and suggest solutions
4. Suggest 2 "leverage activities" that would have outsized impact
```

### Daily Review Prompt
> Use this at the end of each day:

```
Here's what I planned vs. accomplished today:
- Planned: [list]
- Completed: [list]
- Incomplete: [list]

Please help me:
1. Identify why incomplete items didn't get done
2. Suggest how to prevent this tomorrow
3. Rate my day's effectiveness (1-10) with reasoning
4. Suggest one adjustment for tomorrow
```

### Weekly Reflection Prompt
> Use this every Friday:

```
Week in review:
- Goals achieved: [list]
- Goals missed: [list]
- Unexpected wins: [list]
- Biggest challenge: [describe]

Please help me:
1. Identify patterns in what I accomplished vs. missed
2. Suggest process improvements for next week
3. Recommend 3 goals for next week based on momentum
4. Rate this week's overall productivity and well-being
```
